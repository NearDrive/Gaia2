# Decisions i principis no negociables

## Principis (checklist)
- [ ] NEAT és obligatori.
- [ ] No hi ha categories semàntiques hardcoded; la NN rep embeddings (no “water/food/enemy”).
- [ ] Headless primer, viewer després.
- [ ] Determinisme i seed com a prioritat per reproduir runs.
- [ ] PRs petits i CI verd com a gate.
- [ ] Tipatge fort + nullable enable.

## Què NO farem en MVP
- [ ] Viewer o UI.
- [ ] Simulació complexa o features de joc.
- [ ] Persistència real.
- [ ] Categories semàntiques hardcoded.
- [ ] Dependències exòtiques sense justificació explícita.
