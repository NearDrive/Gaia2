# Roadmap Codex

## Fase 0 — Preparació
- Documentació base i principis no negociables.
- Guardrails d’estil i CI.

## Fase 1 — Nucli headless
- Pipeline d’execució determinista.
- Entrades basades en embeddings (sense semàntica hardcoded).
- Integració bàsica amb NEAT.

## Fase 2 — Observabilitat
- Logs mínims i mètriques.
- Exportacions simples per inspecció.

## Fase 3 — Viewer
- Visualització separada del headless.
- Instruments de depuració visuals.

## Nota
Aquest roadmap és orientatiu i s’actualitza amb decisions explícites a docs/DECISIONS.md.
